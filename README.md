# pubg-mp
Match Placement Prediction for the popular online game: PUBG

## Installation

1. Install Pipenv:
```pip3 install pipenv```
2. Install dependencies:
```pipenv install```
3. Install pre-commit hook:
```./scripts/install-precommit-hook.sh```
4. Copy .env and fill in key
```cp .env.sample .env```

## Running

```pipenv run <command>```
